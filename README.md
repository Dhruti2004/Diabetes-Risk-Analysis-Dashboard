# 🩺 Diabetes Risk Analysis Dashboard (Power BI)

An interactive **Diabetes Risk Analysis Dashboard** built using **Microsoft Power BI** to visualize and analyze key medical indicators such as glucose level, BMI, age, insulin levels, and pregnancy count.
The goal of this project is to transform raw healthcare data into meaningful visual insights that help identify potential diabetes risk patterns.

---

# 📊 Project Overview

Diabetes is a major health concern worldwide. Medical datasets often contain large amounts of patient information that are difficult to analyze using traditional methods.

This project presents a **data visualization dashboard** that helps users understand relationships between various health factors and diabetes outcomes. The dashboard simplifies complex data and enables quick analysis through interactive charts and filters.

---

# 🎯 Objectives

* Analyze diabetes-related healthcare data
* Identify patterns and risk factors
* Visualize patient statistics in an interactive dashboard
* Provide insights for better understanding of diabetes trends
